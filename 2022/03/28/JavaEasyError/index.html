Java易错点

发表于2022-03-28|更新于2022-03-30|Java
|阅读量:

The public type must be defined in its own file

翻译：公共类必须被定义在它自己的文件中

产生错误的原因：

  1. 定义Java类名同文件名不一样

  2. 一个文件中有多个类，只有与文件名一致的类名，才能声明为：public

内部类不能声明为：public

一个文件中只能有一个public类
